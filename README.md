# HR Onboarding Agent (Backend-First)

This project is a high-speed AI agent that automates the generation of HR onboarding summaries using the Gemini 2.5 Flash API.

## 🛠️ Setup Instructions
1. Clone this repository.
2. Run `pip install requests python-dotenv`.
3. Create a `.env` file in the main folder and add `GEMINI_API_KEY=your_key_here`.

## 🚀 How to Run
To see the agent in action, run the core logic from the terminal:
`python ET_Hackathon/brain.py`
