## ⚙️ Quick Start Guide

1. **Install:** `pip install -r requirements.txt`
2. **Setup:** Add your API Key to the `.env` file.
3. **Run:** `streamlit run app.py`

---
## 1️⃣ Install Dependencies

**Step 1:** Run this command in your terminal:

pip install -r requirements.txt

---

## 2️⃣ Configure API Key

**Step 2:** Create a file named **.env** and add your key:

GEMINI_API_KEY=your_actual_api_key_here

---

## 3️⃣ Run the Application

**Step 3:** Start the dashboard with this command:

streamlit run app.py

---

## 4️⃣ Test with Samples

**Step 4:** Upload **james.png** to see the AI agent verify the user in real-time.

---

## 🌟 Key Features
* **⚡ AI Extraction:** Instantly pulls Name, Dept, and Role from IDs.
* **📜 Audit Trail:** Every action is logged in `audit_log.txt`.
* **🎉 UI Celebrations:** Interactive success balloons upon verification.
