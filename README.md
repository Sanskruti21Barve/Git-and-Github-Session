# HR Onboarding Agent (Backend-First)

![image alt](https://github.com/Sanskruti21Barve/Git-and-Github-Session/blob/main/agenticai1.png)

This project is a high-speed AI agent that automates the generation of HR onboarding summaries using the Gemini 2.5 Flash API.

## 🛠️ Setup Instructions
1. Clone this repository.
2. Run `pip install requests python-dotenv`.
3. Create a `.env` file in the main folder and add `GEMINI_API_KEY=your_key_here`.

## 🚀 How to Run

Clone the repository.

Create a .env file in the root folder.

Add your API key: GEMINI_API_KEY=your_key_here.

Run pip install python-dotenv requests.

If you are outside the folder: This command is correct.

If you are inside the folder: You just run python brain.py.
